# Crontinel MVP Task Index

This folder holds the detailed task sheets for the MVP plan.

## How to read it
- Stage files explain the release order.
- Feature files explain the objective and acceptance criteria.
- Task sheets explain the concrete to-do list for each feature.

- `01-repo-baseline-validation.md`

- `08-failure-alerts-and-rules.md`
- `09-package-family-guidance.md`
- `10-release-verification-and-handoff.md`

## Structure
Each task sheet contains:
- Files to inspect
- To do
- Implementation steps
- Acceptance criteria

The task sheets stay compact enough for live use, but they now include enough detail to hand off directly to an implementer.
